# traffic-light-project
Button using for traffic light
 * A1 - button 1 (PA1)
 * A2 - button 2 (PA4)
 * A3 - button 3 (PB0)
Pedestrian mode
 * A0 - button for pedestrian. (PA0)
 * D7 - pedestrian light - red. (PA8)
 * D8 - pedestrian light - green. (PB10)
 * D12 - Pedestrian Buzzer. (PA6)
Traffic light 1
 * D1 - RED. (PA9)
 * D2 - YELLOW. (PA10)
 * D3 - GREEN. (PA11)
Traffic light 1
 * D4 - RED. (PA9)
 * D5 - YELLOW. (PA10)
 * D6 - GREEN. (PA11)
